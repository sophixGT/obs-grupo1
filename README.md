Para esta actividad el formato del documento es el siguiente:

    - Fuente: Arial
    - Tamaño fuente: 11
    - Tamaño fuente Titulo 1: 18
    - Tamaño fuente Titulo 2: 14

**Al finalizar, actualizar índice.**
Son máximo 2 páginas para cada ejemplo.

La estructura es la siguiente:

    Aplicación # – Nombre de la aplicación.

    1. Descubrimiento de la aplicación:
    - Motivación de la aplicación
    - Número de usuarios / departamentos
    - Tipología de usuario
    - Frecuencia de uso
    - Quién la desarrolló
    - Estado de la aplicación
    - Recursos requeridos
    2. Métricas relevantes
    3. Evaluación y decisión (6 R's)
    4. Beneficios
    5. Riesgos
    6. Evaluación numérica
    7. Conclusión
